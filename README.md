# Amp Manager


Desktop app for local development. Docker-based stack with a click-and-run GUI. No terminal required. 


## ⌨️ Prefer the Terminal?

Try [AMP CLI](https://github.com/amp-manager/amp-cli), a command-line version 
with the same educational stack and full scripting control.

→ Both tools are open, free, and designed to help you learn how a dev stack works.

## ⚠️ Windows SmartScreen Warning

When you first run AMP Manager, Windows may show:

> "Windows protected your PC"

This is normal for unsigned apps. To proceed:

1. Click **More info**
2. Click **Run anyway**

AMP Manager is open-source and safe. [View Source Code](https://github.com/amp-manager/amp-manager)
